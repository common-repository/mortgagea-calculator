== Description ==

**The Mortgage Calculator helps estimate the monthly payment due along with other financial costs associated with mortgages.**

**For using This plugin you Can Know What kind of interest rate you can get? What kind of payment should you expect?.**


To run this plugin you need to put shortcode in your page:
([wmc_calc])


<h3>Features:</h3>

<ul>
<li>Easy to use</li>
<li>Mobile friendly ui</li>
<li>Change calculator background color</li>
<li>Customize header title text and font size</li>
<li>Also customize header border color</li>
<li>Customize form input field title color, background hover color</li>
<li>Display result with chart</li>
<li>Select chart type</li>
<li>Change chart monthley payment, total payment and total interset color</li>
<li>Customize schedule Sttting</li>
<li>Enable/ Disable annual and monthly schedule</li>
<li>Customize header title text</li>
<li>Customize result text setting</li>
<li>Customize result chart text setting</li>
<li>Set min and max purchase price with default value</li>
<li>Set min and max rate of interest with default rate</li>
<li>Set min and max loan term with default year</li>
</ul>
